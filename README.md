# mu-transmutation-glueball

**The Scale Sector: mu from Dimensional Transmutation and Glueball Dark Matter**

Mirror-QCD transmutation origin of the 28.5 MeV scale, glueball SIMP dark matter, the Lambda_h ~ 52 MeV pincer (UZ coefficient, cluster SIDM, LRD seeding).

Part of the USC research program (SEDE / ECCG / APDM unification).

## Status
Pre-submission working repository, seeded 2026-07-17 from a private research hub;
full research history (including audit trails and
retractions) lives there.
